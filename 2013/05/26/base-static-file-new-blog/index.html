基于静态文件的新博客

周五浏览网页时偶然看到别人用 Liquidluck 搭建的静态博客站点：利用 GitHub 的webhook功能进行自动部署，同时支持 Markdown 文件，可将其自动输出为静态文件，此外还支持一些博客常见的功能，如标签、分类、归档等，非常吸引人。

于是，周末两天折腾，部署上之后，自己还做了套主题 liquidluck-theme-moodpo ，就是现在这个样子。期间遇到各种问题，趁还没忘记录下来。
